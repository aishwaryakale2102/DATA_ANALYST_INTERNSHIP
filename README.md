# DATA_ANALYST_INTERNSHIP

# Data Cleaning and Preprocessing Task-1

## Objective
Clean and preprocess a raw dataset with null values, duplicates, inconsistent text values, incorrect date formats, and datatype issues.

## Dataset Used
Netflix Movies and TV Shows Dataset

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Cleaning Steps Performed
1. Checked missing values
2. Filled missing values in director, cast, country, rating, and date_added
3. Converted date_added into datetime format
4. Renamed columns into lowercase with underscores
5. Standardized text values
6. Checked and corrected data types
7. Checked duplicate rows
8. Saved cleaned dataset

## Summary
Removed duplicate rows, handled missing values, standardized text values, converted date formats, renamed column names, and corrected data types.

## Output
Generated cleaned dataset file: cleaned_netflix_titles.csv
